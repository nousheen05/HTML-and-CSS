Q) What is HTML?
- Hyper Text Markup Language
- not a programming language(donot have any kind of logics involved, not making it work on itself)
- its extremely important and one of the most important base for web dev
- hyper text => text containing links that direct to other pages
- markup - containing set of tags/elements to define structure of web page
